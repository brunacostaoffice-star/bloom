# Bloom

Protótipo PWA mobile-first do app de treino da Bruna.

## Visualizar localmente

Abra `index.html` no navegador. Para testar instalação/offline, use um servidor local:

```bash
python3 -m http.server 3000
```

Depois acesse `http://localhost:3000`.

## Vercel

Suba toda a pasta como projeto estático.
